App Permissions

The Application requests the following permissions which are strictly necessary to provide its core document scanning and PDF creation functionality:

  • CAMERA — This permission is essential to allow users to scan physical documents directly using their device camera. Without this permission, the core scanning feature of the app cannot function. The camera is only activated when the user explicitly initiates a scan and is not used for any other purpose.
  • READ_EXTERNAL_STORAGE — This permission is required to import existing images or files stored on your device for PDF conversion. It is used solely to allow users to select files they wish to convert and is not used to collect or transmit your files.
  • Photo / Video Access — This permission is required to allow users to select images from their gallery for scanning and PDF creation. Access is only granted when the user actively chooses to pick an image and is not used to browse or collect your media.

These permissions are used solely for the stated core app functionality. No personal data is collected beyond what is necessary for the app to work.
