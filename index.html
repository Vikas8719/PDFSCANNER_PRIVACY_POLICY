App Permissions

The Application requests the following permissions. Each is strictly necessary for core functionality:

Permission Data Type Accessed Purpose
CAMERA Camera To scan physical documents using your device camera. Only activated when the user explicitly starts a scan. Not used for any other purpose.
READ_EXTERNAL_STORAGE Photos, Videos, Files, Audio To import existing images or files from your device for PDF conversion. Only files the user explicitly selects are accessed. No files are transmitted.
WRITE_EXTERNAL_STORAGE Photos, Videos, Files, Audio To save generated PDF files to your device storage. Used only to write the output PDF that the user creates. No personal media is modified.
READ_MEDIA_IMAGES Photos To allow selection of images from your gallery on Android 13+. Only images the user explicitly selects are accessed.
INTERNET Network Required to load advertisements via AdMob and send anonymous crash/analytics data to improve the app.
ACCESS_NETWORK_STATE Network status To check network connectivity before loading ads or sending analytics data.
FOREGROUND_SERVICE None To keep PDF processing running in the foreground when converting large documents, so the operation is not interrupted.
WAKE_LOCK None To prevent the device screen from sleeping during long document scans or PDF conversions.
AD_ID (com.google.android.gms.permission.AD_ID) Device or Other IDs To access the Advertising ID for serving personalized ads via AdMob.
ACCESS_ADSERVICES_AD_ID Device or Other IDs Used by Google AdServices SDK for ad attribution and measurement on Android 13+.
ACCESS_ADSERVICES_ATTRIBUTION Attribution data Used by Google AdServices SDK to measure ad campaign effectiveness. No personal data is shared beyond what AdMob requires.
ACCESS_ADSERVICES_TOPICS Interest-based topics Used by Google AdServices SDK to serve interest-based ads using Android's Privacy Sandbox Topics API, which runs on-device and does not share browsing history with third parties.

No personal data is collected through these permissions beyond what is necessary for the stated core app functionality.
